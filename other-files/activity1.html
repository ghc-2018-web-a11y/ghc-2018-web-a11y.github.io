Activity 1

Set up

  1. Download workshop materials here, if you haven't already.
  2. Open the workshop materials folder and navigate to the website/ directory.
  3. Open index.html in Chrome. NOTE: If you already feel comfortable using a screen reader on your machine, open index_no_screenreader.html instead.

If you're using index.html (the version with the embedded screenreader) you'll notice a line of buttons on the bottom of the screen. Normal screen readers include keyboard shortcuts you can use to navigate a page. The embedded screen reader exposes these shortcuts as buttons, such as the “Next”, “Previous”, and “Heading” buttons at the bottom of the page. Use tab and shift+tab to navigate to the next or previous interactive element. Use the left and right buttons at the bottom of the page to navigate other elements, such as text.

Exercises

Complete the following to explore the webpage. Just explore for now instead of trying to debug; we'll do that in the next activity!

  1. Use the "Heading" button on the screen reader to jump to all the headings in the app. Are you able to use this button to jump to the very first site title heading on the page?
  2. Use the keyboard (tab, shift+tab, and the enter key) to navigate to the "Buy" dialog for INSERT ITEM HERE item
  3. You'll notice that the tab key only goes through some elements, such as buttons. To navigate through non-interactive elements such as text nodes, you'll have to use screenreader shortcuts (simulated on our site by the headings, next, and previous buttons at the bottom of your screen). Try using these buttons to navigate to INSERT ITEM HERE.
  4. Try using the keyboard (tab and enter key) to buy an item.
  5. Click the first "buy" button using the mouse. You'll notice a dialog appear. Without looking at the screen, did you receive any indication from the screen reader that a dialog popped up? Do you know what the dialog is for?
  6. Using the keyboard, try to tab to the first input field in the dialog.
  7. Once you've tabbed to the input fields in the dialog, try filling out the fields. Looking at the screen, you can determine that the input fields are for quantity, name, and email. Without looking at the screen, can you figure out what each input field is for?
  8. Try navigating to the "X" button on the dialog.
  9. Try selecting the "submit" button.
